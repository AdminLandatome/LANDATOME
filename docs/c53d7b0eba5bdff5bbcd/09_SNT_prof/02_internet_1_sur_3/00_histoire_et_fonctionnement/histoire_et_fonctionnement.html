Histoire et Fonctionnement d'Internet

Origines dans les années 1960 avec ARPANET

  • ARPANET : Projet lancé en 1969 par l'ARPA aux États-Unis pour créer un réseau de communication résistant aux attaques militaires.
  • Première connexion entre l'UCLA et le Stanford Research Institute.

Évolution dans les années 1980 et 1990 avec TCP/IP et le Web

  • Protocole TCP/IP : Norme pour les communications sur ARPANET dans les années 1980.
  • World Wide Web : Inventé par Tim Berners-Lee en 1989.
  • Popularisation avec les navigateurs web comme Mosaic en 1993.

Expansion mondiale d'Internet et sa démocratisation

  • Années 1990 et 2000 : Croissance exponentielle de l'utilisation d'Internet.
  • Applications web, moteurs de recherche, et plateformes de communication ont contribué à la démocratisation.
  • Aujourd'hui : Internet est essentiel pour les communications, l'éducation, le commerce, et les loisirs.

Internet comme réseau de réseaux

  • Définition : Un "réseau de réseaux" constitué de millions de réseaux interconnectés.
  • Interconnexion via des points d'échange Internet (IXP).
  • Utilisation de protocoles standards pour communiquer.

Concepts de nœuds, de routeurs et de serveurs

  • Nœuds : Dispositifs connectés à Internet (ordinateurs, smartphones, IoT).
  • Routeurs : Dirigent le trafic de données sur Internet.
  • Serveurs : Ordinateurs spécialisés fournissant des services (hébergement de sites web, etc.).

Différence entre Internet et le Web

  • Internet : Infrastructure physique (câbles, routeurs, serveurs).
  • Web : Système de documents hypertextes accessibles via Internet.
  • Utilise les protocoles HTTP et HTTPS, les langages HTML et CSS.
