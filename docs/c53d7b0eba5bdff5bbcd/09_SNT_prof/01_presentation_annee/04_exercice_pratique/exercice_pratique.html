Exercice Pratique : Recherche et Documentation

Objectif

Cet exercice a pour but de vous familiariser avec l'utilisation d'un navigateur web pour rechercher des informations et les documenter dans un fichier texte.

Instructions

  1. Ouvrir un navigateur web :

    Lancez le navigateur web de votre choix (par exemple, Google Chrome, Mozilla Firefox, Microsoft Edge, Safari).

  2. Rechercher une information :

    Effectuez une recherche sur le sujet suivant : "L'histoire de l'Internet".

    Trouvez une source fiable (comme un site éducatif, un article de Wikipedia, ou une publication d'une organisation reconnue).

  3. Créer un document texte et y inscrire des informations :

    Ouvrez un logiciel de traitement de texte (par exemple, Microsoft Word, Google Docs, LibreOffice Writer, ou même le Bloc-notes).

    Créez un nouveau document texte.

    Dans ce document, inscrivez les informations suivantes :

    • Un titre : "L'Histoire de l'Internet"
    • Un résumé de 100 à 150 mots sur l'histoire de l'Internet basé sur vos recherches.
    • Citez la source de vos informations à la fin du document.

Exigences

  • Le résumé doit être clair et concis, couvrant les points clés de l'évolution de l'Internet.
  • La source d'information doit être citée correctement (inclure l'URL et le nom du site).
  • Sauvegardez le document sous le nom "00E01_2H_NOM_PRENOM" (votre enseignant vous indiquera le lieu de sauvegarde)

Exemple de document attendu

Titre : L'Histoire de l'Internet

Résumé :
L'Internet a commencé comme un projet de recherche militaire aux États-Unis dans les années 1960 avec la création de l'ARPANET. Au fil des décennies, il a évolué pour devenir un réseau mondial interconnecté. L'invention du World Wide Web par Tim Berners-Lee en 1989 a transformé l'Internet en un outil accessible au grand public, facilitant la navigation et l'accès à une vaste quantité d'informations. Aujourd'hui, l'Internet est indispensable dans de nombreux aspects de la vie quotidienne, du commerce électronique à la communication instantanée.

Source :
Wikipedia, "History of the Internet", https://fr.wikipedia.org/wiki/Histoire_d%27Internet

Remarques

  • Assurez-vous de vérifier l'exactitude des informations trouvées.
  • Utilisez des outils de correction orthographique et grammaticale pour garantir la qualité de votre texte.
  • Enregistrez et sauvegardez votre document régulièrement pour éviter toute perte de données.
