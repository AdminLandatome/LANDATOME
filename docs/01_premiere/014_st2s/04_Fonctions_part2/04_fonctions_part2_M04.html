EXERCICE N°3 (Application ST2S)

Lors d'une épidémie, on modélise le nombre de malades par la fonction \(f\) définie sur \([0 ; 12]\) par :

\(f(x) = -20x^2 + 240x + 100\)

où \(x\) est le nombre de semaines écoulées et \(f(x)\) le nombre de malades pour 100 000 habitants.

  1. Calculer la fonction dérivée \(f'(x)\).
  2. À quel moment (en semaines) le pic de l'épidémie est-il atteint ?
  3. Pendant combien de semaines y-a-t-il eu au moins 500 malades ?.
  4. Quel est le nombre maximal de malades atteint selon ce modèle ?

FONCTIONS PART2 M04C (Corrigé)

Correction Exercice 1

1) La droite \((d)\) a pour coefficient directeur 3. On cherche donc si \(f'(x) = 3\).
\(f'(x) = x^2 - 2x\). On résout \(x^2 - 2x = 3 \Leftrightarrow x^2 - 2x - 3 = 0\).
Les solutions sont \(x = -1\) et \(x = 3\). Oui, il existe deux tangentes parallèles à \((d)\).

2) Coordonnées :
Pour \(x = -1\), \(f(-1) = -4/3\). Point \(A(-1 ; -4/3)\).
Pour \(x = 3\), \(f(3) = 0\). Point \(B(3 ; 0)\).

RETOUR À L'EXERCICE 1

Correction Exercice 2

On a \(f'(x) = -4x + 8\).

  1. \(-4x + 8 = 4 \Rightarrow -4x = -4 \Rightarrow x = 1\). Point \((1 ; f(1)) = (1 ; 1)\).
  2. \(-4x + 8 = -2 \Rightarrow -4x = -10 \Rightarrow x = 2,5\). Point \((2,5 ; f(2,5)) = (2,5 ; 2,5)\).
RETOUR À L'EXERCICE 2

Correction Exercice 3

  1. \(f'(x) = -40x + 240\).
  2. Le pic est atteint quand \(f'(x) = 0\) : \(-40x + 240 = 0 \Rightarrow 40x = 240 \Rightarrow x = 6\). Le pic est atteint au bout de 6 semaines.
  3. Graphiquement \(f(x)\geq 500\) admet comme ensemble des solutions \([2\;;\;10]\).
    On en déduit qu'il y a eu au moins 500 malades pendant 8 semaines.
  4. Le nombre maximal de malades est \(f(6) = -20(6^2) + 240(6) + 100 = -720 + 1440 + 100 = 820\).
RETOUR À L'EXERCICE 3
