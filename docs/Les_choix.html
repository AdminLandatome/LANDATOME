Pour choisir un automatisme, il suffit de cliquer sur l'image correspondante.
Les menus se déroulent et se rétractent en cliquant dessus.
Vous devez choisir des séries de cinq automatismes.
Pour utiliser les séries, on double-clique sur "diapo-titre"(.html) puis sur la ligne grisée horizontale la plus haute pour aller à la diapo suivante.
(Un exemple est disponible ICI avec le zip là )
Attention, si la dernière série ne comporte pas 5 éléments alors elle sera ignorée.
